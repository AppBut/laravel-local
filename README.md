# Laravel-Local

[![Software License][ico-license]](LICENSE.md)
[![Total Downloads][ico-downloads]][link-downloads]

Provides easy setup for local dev environments with certain tools already setup

## Install

Via Composer

``` bash
$ composer require appbut/laravel-local
```
